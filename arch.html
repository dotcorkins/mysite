Arch Linux Install Guide 2024

Arch linux is a cool distro. Lets install it.
[source]
 I reccomend just reading this. MUCH better.

This guide may pertain programs specific to windows. Like rufus USB flashing. But if you're on linux already, I assume you know how to flash USB's on linux.

First you have to acquire a archlinux ISO. Scroll down to the mirrors and select the one you want. I always go with arizona state's mirror. After this (if on windows) download the rufus program. Open rufus and plug in your USB. I reccomend this USB to be a minimum of 8gb in size. Then select the ISO and flash it to the drive. Now, go into your BIOS and make sure secure boot is disabled. Reboot into windows, plug the USB in. Boot into BIOS and make sure the USB is first in the boot priority. Hopefully this works. You should be booted into the live USB. Connect to the internet. How? using the iwctl utility. run
# iwctl
then run
# iwctl station list
